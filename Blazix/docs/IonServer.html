    desisoft.dsap.IonServer interface

    This interface is optionaly implemented by Ion server objects.  It can be used to
    • detect when a new connection is made
    • detect when a connection is closed.
    • get a callback object.
    public interface IonServer
{
    void onConnect();
    void setCallback( Object callbackObject );
    void onClose();

}
    If the server object implements this interface, it will receive notification upon connection and upon connection termination, as well as a callback object if the client did the lookup with a callback interface.  Any calls made using the callback object will be given to the client to process.
