    The Blazix tag library

    Blazix provides a JSP 1.1 compliant tag library for easy JSP development.

    To use the tag library, add the line 

    <%@ taglib prefix="blx" uri="/blx.tld" %>
    to the JSP page.  (This line is not required, but recommended.  See notes.)

    This makes available the following tags.  Follow the links below for detailed explanations of the tags.

    Form Processing Tags

    blx:setProperty
blx:getProperty

    Email Tags

    blx:email
blx:emailTo
blx:emailCc
blx:emailBcc
blx:emailTo
blx:emailSubject
blx:emailAttach

    SQL tags

    blx:sqlConnection
blx:sqlQuery
blx:sqlExecuteQuery
blx:sqlGet
blx:sqlExecuteUpdate

    Internationalization Tags

    blx:xlt
blx:xltFile
    Notes:
    • Any prefix may be be used, but blx is the recommended prefix.
    • Blazix uses the "well-known URI" mechanism of JSP 1.1 to locate /blx.tld if it is not available through the standard mechanism.  As per the recommendations in JSP 1.1 specifications, the "well-known URI" mechanism should not be depended upon.  The Blazix installation includes "blx.tld" in the "redist" directory.  When packaging the application, the "blx.tld" file should be packaged and added to web.xml.
