    Installation

    To install Blazix Java Server on a Windows machine, run the self-installing executable.

    Requirements:  Java should be available in the path -- typing java in a command window should start the java interpreter.  If it does not, you need to make sure set your PATH is set correctly.  For many features, javac is also required.

    Testing the installation:  Run "Blazix Web Server" from the Start menu.

    This will start the Blazix web server.  When you see the message "Blazix Web Server operational", visit the URL http://DESKTOP-NKM5MK3/ or http://localhost/ in your browser.

    This should bring you to the default initial web page, which contains further links for testing the installation.

    If the server doesn't start up correctly, running it in a command window might make it easier to see any error messages produced.  The server also produces a log file, which can be reviewed for any error messages.

    Changing PATH and CLASSPATH:  For development environments, it is recommended that you change CLASSPATH to include C:\Blazix\blazix.jar in it, and change PATH to include C:\Blazix in it.   (The commands are set up by the installer such that changing PATH or CLASSPATH is not required, but the full set of tools will not be accessible -- including the EJB compiler blxejbc or the jspDebug tool --  without changing PATH and CLASSPATH.)

    If you are not familiar with changing PATH and CLASSPATH, see our brief introduction to setting PATH and CLASSPATH variables.
