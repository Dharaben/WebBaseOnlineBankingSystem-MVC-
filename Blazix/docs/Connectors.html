    Adding connectors for iPlanet/Netscape, Apache or IIS servers

    Blazix is designed to run by itself as a complete web-server.

    Depending upon client requirements, Blazix can also be used as a Java Application Server with other servers such as iPlanet/Netscape, Apache or IIS servers running at the front end.

    This requires aditional software called connectors.

    Connectors and instructions to install them are located at http://www.blazix.com/connectors.html
